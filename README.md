## 프로젝트 소개
**건강해GYM 프로젝트**는 `Spring Boot`와 `React`를 기반으로 **헬스 커뮤니티** 서비스를 구축한 프로젝트 입니다.

## 🕰️ 개발 기간
- 23.04.10 - 23.05.18 (6주)
